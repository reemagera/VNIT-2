# VNIT-2
The Flaskapp.rar file is the integration of the front end part. Here, python and HTML files are to be integrated.
HTML file in the template contains all the UI components
sample.py has the main program for creating charts in python
app.py is the main application for flask. 
This is run by using 'flask run' directory. This command renders the UI where the user can enter the symbol in the search bar.
After this is done, by 'sample.py' in CLI, the user gets to analyse the data according to their wish.
The search history could be found in the 'history.csv' file
